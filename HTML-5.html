Section 0: Setting up a page

Declare the Doctype> of an HTML page at the beginning of your code. You will do this for every HTML document you set up. Always close all HTML in the html tag.

Use a head element to tag your title. Use a main element to tag the main body. Tag with a div element to contain elements you want to divide from others. The footer element is for the bottom of the page.

Section 2: Interactive Elements

In this section, we'll cover several input types. Input elements inside of form tags allow someone browsing your web page to interact with it by sending it data.

Section 2.1: Input Box

You can make a box where a user can type and send their response to the server hosting your website. We'll do this by creating an input field and adding a "submit" button to it.

If you want to make an input box required (like if you're creating a page with a form and someone needs to fill out their name), just add "required" to the input tag.
